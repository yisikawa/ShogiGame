# 将棋ゲーム

ブラウザで動作する将棋ゲームです。JavaScriptで実装されており、人間対人間、人間対AI、AI対AIのモードでプレイできます。

## 機能

- 9x9の将棋盤で将棋をプレイ
- 人間対人間モード
- 人間対AIモード（初級・中級・上級）
- AI対AIモード
- 持ち駒の表示（後手は左側、先手は右側）
- 棋譜の記録と再生
- 成りの選択
- 合法手のハイライト表示

## 使い方

1. `index.html`をブラウザで開く
2. プレイモードを選択
3. 駒をクリックして選択し、移動先をクリックして移動

## ファイル構成

- `index.html` - HTML構造
- `game.js` - ゲームロジックとAI
- `style.css` - スタイルシート
